Portfolio

Tic-Tac-Toe Game

This project game allows two players to play a classic game of Tic-Tac-Toe on the computer. It displays a 3x3 grid and prompts players to take turns to enter their moves by specifying the row and column where they want to place their symbol (X or O). The program checks for valid moves, updates the grid, and checks for a win or a draw after each move. It continues until one player wins or the game ends in a draw. The program provides clear instructions and can display the outcome of the game.

To-Do List App

This project app helps users create, manage, and organize their to-do lists. Users can add tasks, set deadlines, mark tasks as completed, and remove tasks. It provides a basic interface for staying on top of tasks and managing daily activities efficiently.
